## 二、YOLO 模型参数说明

### 2.1 模型加载参数

```python
from ultralytics import YOLO

model = YOLO("models/yolo11n.pt")

# 推理参数（默认值）
results = model.predict(
    source=image_path,       # 输入源：图片路径 / numpy 数组
    conf=0.25,               # 置信度阈值（低于此值的检测被过滤）
    iou=0.7,                 # NMS IoU 阈值（高于此值的重叠框被合并）
    imgsz=640,               # 推理图像尺寸
    device="cpu",            # 推理设备
    verbose=False            # 不打印冗余日志
)
```

### 2.2 COCO 类别（YOLO11 预训练，80 类）

| class_id | 类别名 | 本项目关注 |
|---|---|---|
| 0 | person | ✅ 风险类别（默认） |
| 1 | bicycle | - |
| 2 | car | - |
| 3 | motorcycle | - |
| ... | ... | - |

> 可通过 `model.names` 获取完整映射字典，本项目默认仅将 `person` 设为风险类别。

### 2.3 检测结果结构

```python
# results[0].boxes 属性说明
boxes.cls    # Tensor[N]  类别 ID
boxes.conf   # Tensor[N]  置信度 [0, 1]
boxes.xyxy   # Tensor[N, 4] 边界框 [x1, y1, x2, y2]（左上角 + 右下角）
boxes.xywh   # Tensor[N, 4] 边界框 [cx, cy, w, h]（中心点 + 宽高）
```

### 2.4 Detector 单例封装参数

| 参数 | 默认值 | 说明 |
|---|---|---|
| `sample_interval` | 1.0 秒 | 视频采样间隔 |
| `max_evidence_frames` | 10 | 最多保存的证据帧数 |
| `model_path` | `models/yolo11n.pt` | 模型文件路径 |

### 2.5 审核规则配置参数

```json
{
    "risk_classes": ["person"],     // 风险类别列表（COCO 类别名）
    "reject_confidence": 0.60,      // 拒绝阈值：>= 此值 → reject
    "review_confidence": 0.35,      // 复核阈值：>= 此值 → review
    "min_evidence_frames": 1        // 最少证据帧数
}
```
